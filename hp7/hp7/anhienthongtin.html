HTML là ngôn ngữ đánh dấu được thiết kế để tạo các trang web trên World Wide Web. Nó được hỗ trợ bởi CSS để định dạng giao diện và JavaScript để thêm các tính năng tương tác.

CSS được sử dụng để thiết kế giao diện trang web, bao gồm màu sắc, phông chữ, và cách bố trí nội dung.

JavaScript là ngôn ngữ lập trình để thêm các tính năng tương tác vào trang web, như xử lý sự kiện và cập nhật nội dung động.
